Política de Privacidad - Konnexia

Esta Política de Privacidad corresponde a Konnexia, una plataforma utilizada por empresas para gestionar la comunicación con sus clientes mediante la API oficial de WhatsApp Business de Meta.

Konnexia accede únicamente a la información necesaria para proporcionar las funcionalidades autorizadas, como el envío y recepción de mensajes de WhatsApp Business y la administración de conversaciones.

La información obtenida mediante la API de Meta se utiliza únicamente para el funcionamiento de la plataforma. No se vende ni se comparte con terceros, salvo cuando sea necesario para prestar el servicio o cuando la legislación aplicable lo requiera.
